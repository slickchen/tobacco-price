# 📊 商品价格查询工具

一个用于学术研究的移动端商品价格数据查询与分析工具。

## ⚠️ 声明

本项目仅供**学术研究与数据分析**用途，所有价格数据均为历史参考数据，不构成任何交易建议。

## 📱 在线访问

👉 **[点击访问](https://slickchen.github.io/tobacco-price/)**

## ✨ 功能特点

- 🔍 **实时搜索** - 输入关键词快速查找
- 🏷️ **品牌筛选** - 一键筛选各品牌
- 📊 **盈亏分析** - 红色上涨 / 绿色下跌
- 💰 **自定义售价** - 可编辑销售价格，自动计算利润
- 📈 **多维排序** - 价格 / 盈亏 / 利润多维度排序
- 📱 **移动优先** - 完美适配手机浏览
- ⚡ **纯前端** - 无需后端，数据本地持久化
- 💾 **离线保存** - 自定义价格自动存储到本地

## 🚀 部署到 GitHub Pages

### 方法一：Fork 部署（推荐）

1. **Fork 本仓库**
2. **启用 GitHub Pages** → Settings → Pages → Branch: `main` / `root` → Save
3. 访问 `https://你的用户名.github.io/tobacco-price/`

### 方法二：手动创建

1. 新建 GitHub 仓库
2. 上传 `index.html` 文件
3. 启用 GitHub Pages

## 📝 更新数据

编辑 `index.html` 中的 `R` 数组：

```javascript
const R=[
  ["品名", 价格, 盈亏],
  // 添加更多...
];
```

## 📋 数据字段

| 字段 | 说明 | 示例 |
|------|------|------|
| name | 商品名称 | "示例品A" |
| category | 品牌分类 | "分类A" |
| price | 批发参考价 | 208 |
| profit | 盈亏金额 | -10.36 |
| customPrice | 自定义售价 | 220 |
| tar | 含量指标(mg) | 10 |
| retail | 零售参考价 | 250 |
